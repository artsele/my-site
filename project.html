Μελλοντικά Project:



Βασικό Project της εταιρίας αποτελεί η δημιουργία ενός online web browsing παιχνιδιού. Σε αυτο το παιχνίδι ο κάθε νέος χρήστης θα καλείται να δημιουργήσει έναν καινούριο χαρακτήρα και θα έχει την επιλογή να διαμορφώσει μόνος του τον τύπο μάχης του.Επίσης στο παιχνίδι θα υπάρχουν μηχανισμοί όπως:

  1. Μαγαζί όπλων, από το οποίο θα μπορεί ο παίκτης να αναβαθμίζει τον εξοπλισμό μάχης του.
  2. Η Αρένα στην οποία θα έρχονται αντιμέτοποι οι παίκτες μεταξύ τους για να κερδίζουν πόντους δόξας.
  3. Οι κατακόμβες στις οποίες θα πρέπει να συνεργαστούν οι παίκτες για να καταφέρουν να νικήσουν τα πανίσχυρα τέρατα και να επιβραβευθούν με τον θησαυρό τους.
